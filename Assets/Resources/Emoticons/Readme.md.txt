How to Make Material?

1. Image Setting
  1) Download Image
  2) Set Texture Type - Sprite (2D and UI)
2. Material Setting  
  1) Set Albedo - Ready image
  2) Set Shader - Sprites/Default
3. Set Particle Object  
  1) Add Component - Particle System  
  2) Add Component - Particle Controller  
  3) Particle System - Render - Material
